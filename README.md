Ce projet permet de convertir des fichiers en PDF assez simple.

On peut convertir des images en PDF.
On peut aussi convertir des fichiers texte en PDF.
Enfin, on peut fusionner plusieurs fichiers PDF en un seul.


## Installation
Pour installer les dépendances nécessaires, exécutez la commande suivante :

```
git clone https://github.com/Volterborn
cd pdf-converter
```

## Technologies utilisées
- Python 3.13
- Pillow
- FPDF
- PyPDF2

## Comment l'utiliser
Pour pouvoir tester les fonctions, enlevez le # avant la ligne qui permet de tester la fonction.
